Para o entendimento dos códigos postados, e das pastas com as respectivas imagens, seguem instruções:


Base1 e Base2 são uma divisão de todos os lançamentos usados para a análise do D6, estão separadas uma vez que o git não comporta mais 
1000 arquivos em uma única pasta.

D6 é a pasta que contêm os cadastros das faces D6.

Testes1 e Testes2, possuem as 200 imagens usadas para a análise de precisão do reconhecimento de imagem.

Quanto aos códigos:

sift.py é o código que retorna as leituras do reconhecimento.

TesteReconhecimento.py é o código que analisa e retorna a precisão do reconhecimento de imagem.

estat.py faz as análises do lançamento e o teste de chi-quadrado.

Por fim, Kolmogorov-Smirnov.py realiza o teste de aderência e faz sua análise.
